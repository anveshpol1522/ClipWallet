## How to Contribute

We welcome all contributions! Here's how to get started:

1. **Find an issue** — Check the [Issues](https://github.com/shaaravraghu/ClipWallet/issues) page for `good first issue` or `help wanted` tags
2. **Fork & clone** — Fork the repo and clone it locally
3. **Create a branch** — `git checkout -b feature/your-feature`
4. **Initiate an Issue** — Describe the feature you want to add or the bug you want to fix.
5. **Make your changes** — Follow existing code conventions
6. **Submit a PR** — Reference the issue number in your PR description
7. **Add yourself here** — After your PR is merged, add your entry to the table above!
